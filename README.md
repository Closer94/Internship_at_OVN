# Internship at OVN

### Creavorite 서비스 소개
 - 전 세계 디자이너/아티스트/마케터들이 지속적이고 능동적인 경제활동을 할 수 있는 전문화된 디자인/창작 서비스 플랫폼. 디자인/창작이 필요한 고객들과 디자이너/아티스트/마케터들을 연결해 주는 서비스와 디자인/창작 업무를 효율화할 수 있는 서비스를 제공한다.
